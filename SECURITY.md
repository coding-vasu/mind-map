# Security Policy

## Supported Versions

Currently, the following versions of MindMap are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please use **[GitHub Private Security Advisories](https://github.com/vasuvallabh/mind-map/security/advisories/new)** to report vulnerabilities. We will acknowledge your report within 48 hours and provide a timeline for a fix.

### What to include in a report:

- A description of the vulnerability.
- Proof of concept or steps to reproduce.
- Potential impact of the vulnerability.
- Any suggested fixes.

We appreciate your help in keeping this project secure for everyone.
